# План развития MVP Dopamine Office

## Цель

Создать минимальный рабочий вертикальный flow: Home → Core Room → Mission-1.

## Этапы

1. Home
   - Сделать яркий входной экран в стиле Dopamine Office.
   - Кнопка `Начать игру`.
   - Краткий текст про ценности и концепцию.

2. Game / Core Room
   - Сделать хаб с одним интерактивным объектом.
   - Объект должен быть заметным по цвету и форме.
   - При hover менять курсор и визуально выделяться.
   - Текст: `Core Room / выбери зону`.
   - Клик переводит на `/game/mission-1`.

3. Mission-1
   - Заглушка миссии с карточкой и тремя вариантами.
   - Кнопка `назад` на Core Room.
   - Без логики выбора, только UI.

4. FSD
   - `entities/mission` — типы и мок-данные.
   - `features/mission-card` — UI карточки варианта.
   - `shared/ui/button.tsx` и `shared/ui/card.tsx` — переиспользуемые UI-компоненты.

- Добавить ещё 2 миссии.

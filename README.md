# Игра про ценности компании

## Описание

MVP веб-игры, в которой пользователь проходит игровой путь, знакомясь с ключевыми ценностями команды: ответственность, прозрачность и скорость.

## Стек

- Next.js
- TypeScript
- Tailwind CSS
- Feature-Sliced Design

## Запуск проекта

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Архитектура

Проект организован по FSD-принципу. Основные директории в `src`:

- `app/` — маршруты и общий каркас.
- `pages/` — страницы как FSD-кусок для будущей логики.
- `processes/` — игровые потоки и сценарии.
- `features/` — отдельные фичи интерфейса.
- `entities/` — бизнес-сущности игры.
- `shared/` — повторно используемые UI, библиотеки, конфигурации и типы.

## Миссии

1. Ответственность
2. Прозрачность
3. Скорость

## Что реализовано сейчас

- Базовый Next.js + TypeScript проект.
- Tailwind CSS.
- Стартовая страница с кнопкой «Начать игру».
- Страница игры с пустым контейнером для 3D-сцены.
- Начальная FSD-структура.

## Деплой

Проект можно задеплоить на Vercel через стандартный процесс деплоя Next.js.
